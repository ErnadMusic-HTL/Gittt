HTML CheatSheet

1. Components of HTML Elements

Each HTML element consists of the following components:

  • Opening Tag: e.g., <element>
  • Content: The text or other elements inside the tag.
  • Closing Tag: e.g., </element>

Example:

<p>This is a paragraph.</p>

2. Void Elements

Void elements do not have closing tags and are self-contained. They are denoted by their opening tag only.

Examples of void elements include:

  • <img> - for images
  • <br> - for line breaks
  • <hr> - for horizontal rules
  • <input> - for input fields

Example:

<img src="image.jpg" alt="Image Description">

3. Attributes

Attributes provide additional information about an HTML element. They are always included in the opening tag.

Example:

<a href="https://example.com" target="_blank">Link</a>

Common attributes include:

  • id: Unique identifier for an element.
  • class: Class name for styling purposes.
  • style: Inline CSS styles.
  • src: Source URL for images.

4. Correct DOCTYPE Declaration for HTML5

The correct DOCTYPE declaration for HTML5 is:

<!DOCTYPE html>

5. Syntactically Correct Statements

Here are examples of various HTML elements:

  • Article: 
    <article><h2>Article Title</h2><p>This is an article.</p></article>
  • Body: 
    <body><h1>Welcome to My Website</h1></body>
  • Line Break: 
    <p>First line.<br>Second line.</p>
  • Headings: 
    <h1>Main Heading</h1><h2>Subheading</h2><h3>Sub-subheading</h3>
  • Section with Meta-Information: 
    <section><meta name="description" content="This is a section"></section>
  • Top Level Element: 
    <html lang="en"></html>
  • Image Named Team.jpg: 
    <img src="Team.jpg" alt="Team Photo">
  • Paragraph: 
    <p>This is a paragraph.</p>
  • Section: 
    <section><h2>Section Heading</h2><p>Content of the section.</p></section>
  • Label in Title Bar: 
    <title>My Web Page Title</title>
