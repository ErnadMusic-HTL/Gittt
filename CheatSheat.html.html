World Wide Web General

The most essential technologies that made the WWW successful were: HTML, HTTP, Web Browsers, URLs, CSS, JavaScript, Servers.

Resources on the web are addressed using URLs.

A URL is the address used to access a resource on a web server. It contains the scheme, domain name, path, and sometimes a query or fragment.

Web Server

A web server is a software or hardware system that stores, processes, and delivers web content to users over the Internet.

A web server is a computer system that hosts websites and delivers web content to users' browsers over the internet when they request it, typically through the HTTP or HTTPS protocols.

There are multiple ways content can be stored on a web server depending on the type of data and the server's configuration. Some of the ways are: File System (static files like HTML, CSS, JavaScript, images, and videos), Database for Dynamic Content and Data Models, Content Management System (CMS), and Cloud Storage.

Content in web pages is described using several key components and technologies: HTML, CSS, JavaScript, Images, and Multimedia.

Web Client

A web client is a software application or system that interacts with web servers to request and receive web content. The most common type of web client is a web browser.

A web client performs several key functions that enable users to interact with web content. Some of the things they do are: Sends Requests, Receives Responses, Renders Content.

Every web browser is a web client, but not every web client is a web browser. A web client is a broader term that refers to any software application or system that communicates with web servers to request and receive web content. This includes not only web browsers but also other applications that interact with web services. A web browser is a specific type of web client designed primarily for navigating the World Wide Web. It provides a user interface for accessing, rendering, and interacting with web content.

The <head> element is a part of an HTML document that contains important information about the page, but it doesn’t display anything directly on the webpage (title, meta tags, links, scripts).

The <!DOCTYPE html> declaration is a special instruction that tells the web browser which version of HTML the document is using. <!DOCTYPE html> specifies that the document is an HTML5 document, which is the latest version of HTML.

The two topmost elements within the <html> element in an HTML document are: head and body. The head contains the information about the document but doesn’t display anything, whilst the body is the website itself.

The <em> tag is used to define emphasized text. The content inside is typically displayed in italic.

The <strong> tag is used to define text with strong importance. The content inside is typically displayed in bold. You can also use <b> to make things bold but with no importance.
